# Hospital Program Portal

เว็บรวมโปรแกรมโรงพยาบาล พร้อมระบบล็อกอินด้วยเลขบัตรประชาชนจากฐานข้อมูล HOSxP (`hos.opduser`)

## ความสามารถ

- ล็อกอินด้วยเลขบัตรประชาชน 13 หลัก
- ตรวจสอบผู้ใช้จากตาราง `opduser` ถ้าไม่พบจะไม่อนุญาตเข้า
- แสดงข้อความต้อนรับหลังล็อกอิน
- แสดงรายการเมนูโปรแกรมโรงพยาบาล (เช่น FDH Checker)
- แอดมินเพิ่ม/แก้ไข/ลบเมนูได้
- สิทธิ์แอดมินจาก `cid` หรือ `droupname` (ปรับได้ใน `.env`)

## เริ่มใช้งาน

1. ติดตั้ง dependencies
   ```bash
   npm install
   ```

2. ตั้งค่า `.env` จาก `.env.example`

3. รันระบบ
   ```bash
   npm start
   ```

## หมายเหตุ

- ระบบนี้ใช้ไฟล์ `data/menus.json` เก็บเมนูที่แอดมินจัดการ
